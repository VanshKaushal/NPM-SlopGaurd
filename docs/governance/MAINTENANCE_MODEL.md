# MAINTENANCE MODEL RFC

## 1. Abstract
The maintenance model outlines the long-term stewardship of the SlopGuard project to ensure ecosystem longevity and uninterrupted operational stability.

## 2. Stewardship
The core infrastructure will always remain strictly Open Source (MIT). We reject dual-licensing or "open-core" strategies that fragment the deterministic supply-chain guarantees.

## 3. Deprecation Cycles
Features marked for deprecation will emit structured warnings for a minimum of 6 months before being removed in the subsequent major release. This provides enterprise CI pipelines adequate runway to adapt.
